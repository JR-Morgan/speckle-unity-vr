# speckle-unity-vr

A work in progress example project of Speckle in VR!

Currently working with OpenXR on Windows, with the Quest 2.

https://drive.google.com/file/d/1ls5zovY15Bxhis9e0SgxjKp9k_rLsxzE/preview
